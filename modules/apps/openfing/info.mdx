### Context

When I first learned React, I built a new web client for the OpenFING project at the University of the Republic, Uruguay. A few months later, my implementation became the project's official web client, which I maintained and improved for about 4 years.

Originally I used React, PostgreSQL, GraphQL, and Fluent UI. Later, I rewrote it with Next.js, Supabase, Shadcn UI, and Tailwind CSS, since I wanted to simplify the stack and make it easier to maintain.

This site is no longer the official web client for the project.
